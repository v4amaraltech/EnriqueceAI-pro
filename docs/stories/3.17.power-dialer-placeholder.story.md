## Story

**As a** SDR,
**I want** uma tab Power Dialer na tela de execução com UI de fila de discagem,
**so that** eu possa visualizar como será o fluxo de discagem automática quando a integração VoIP estiver disponível.

## Scope

**IN:**
- Tab Power Dialer na tela de execução
- UI de fila de discagem com controles visuais
- Banner "Em breve"
- Pode ser feature flag desabilitada

**OUT:**
- Integração VoIP real (Twilio/Vonage)
- Discagem automática
- Gravação de chamadas
- Transferência de chamadas
- Qualquer backend de telefonia

## Acceptance Criteria

1. Tab "Power Dialer" visível na tela de execução (ao lado de "Execução")
2. Conteúdo da tab: banner "Em breve — Integração com provedor VoIP"
3. UI mockup: lista de leads para discagem com avatar + nome + telefone
4. Controles visuais (não funcionais): Play, Pause, Skip
5. Status placeholder por lead: "Aguardando", "Em chamada", "Concluído"
6. Feature flag: pode ser ocultada via config se necessário
