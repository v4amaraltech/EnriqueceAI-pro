## Status

**Ready**

## Complexity Estimate

**M (5 points)** — Instalação de libs (html2canvas + jsPDF), rendering em div oculta com tema forçado, layout A4, integração com deltas da 5.2.

## Risks

- **R1:** html2canvas pode não capturar CSS vars corretamente — mitigado: forçar tema claro na div de captura com classes inline
- **R2:** Tabelas longas podem ser cortadas na página — mitigado: paginação automática no jsPDF ou split de tabelas
- **R3:** Bundle size aumenta com html2canvas (~400KB) — mitigado: dynamic import para lazy loading

## Dependencies

- **Story 5.2** — Comparação Período-a-Período (deltas devem aparecer no PDF quando comparação ativa)

## Story

**As a** manager,
**I want** to export reports as a professional PDF document,
**so that** I can share performance data with stakeholders who don't have platform access.

## Scope

**IN:**
- Botão "Exportar PDF" ao lado do "Exportar CSV" existente em `ReportsView`
- Geração client-side com `html2canvas` + `jsPDF`
- Layout PDF: header com nome da org + período, tabelas formatadas, footer com data de geração
- Exportar cada tab como seção no PDF (Overall, Cadence, SDR)
- Loading state durante geração ("Gerando PDF...")
- Nome do arquivo: `relatorio-{tipo}-{from}-{to}.pdf`
- Suporte a tema claro para PDF (forçar light mode no snapshot)
- Se comparação ativa (5.2), deltas aparecem no PDF

**OUT:**
- Exportação de Statistics (escopo futuro — muitos charts complexos)
- Agendamento de exportação automática
- Exportação server-side (Puppeteer)
- Customização de layout pelo usuário
- Logo customizado por organização

## Acceptance Criteria

1. Botão "Exportar PDF" visível em `ReportsView`
2. PDF gerado contém header com nome da org + período selecionado
3. Seção "Visão Geral" com funil e métricas
4. Seção "Por Cadência" com tabela de métricas por cadência
5. Seção "Por SDR" com tabela de métricas por SDR
6. Se comparação ativa (5.2), deltas aparecem no PDF
7. Footer com timestamp de geração
8. Loading state exibido durante geração
9. PDF legível em A4 landscape, tabelas não cortadas
10. `pnpm typecheck`, `pnpm lint`, `pnpm test:run` e `pnpm build` passam

## Dev Notes

**Padrão de dynamic import para reduzir bundle:**
```typescript
async function exportToPdf(element: HTMLElement, filename: string) {
  const [{ default: html2canvas }, { default: jsPDF }] = await Promise.all([
    import('html2canvas'),
    import('jspdf'),
  ]);
  const canvas = await html2canvas(element, { scale: 2, useCORS: true });
  const pdf = new jsPDF('landscape', 'mm', 'a4');
  // ... add canvas to pdf
}
```

**Forçar tema claro na div de captura:**
```tsx
<div ref={pdfRef} className="fixed -left-[9999px] bg-white text-black">
  {/* Layout de impressão com classes explícitas */}
</div>
```
