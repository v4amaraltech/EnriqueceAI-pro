## Story

**As a** manager,
**I want** to see my invoice history, payment method, and clear feedback after checkout,
**so that** I have full visibility into my billing and feel confident about my subscription.

## Scope

**IN:**
- Seção "Histórico de Faturas" na billing page (via Stripe API)
- Exibição do método de pagamento atual (tipo + últimos 4 dígitos)
- Banner/toast de sucesso após checkout (`?success=true`)
- Banner/toast de cancelamento após checkout (`?canceled=true`)
- Empty states adequados para trial (sem faturas, sem pagamento)
- Link para Stripe Customer Portal

**OUT:**
- Customização de método de pagamento fora do Stripe Portal
- Boleto/PIX como métodos de pagamento
- Nota fiscal brasileira (NF-e)
- Download de faturas no app (usa PDF link do Stripe)

## Acceptance Criteria

1. Seção "Histórico de Faturas" mostra últimas 10 faturas com: data, valor (BRL), status (pago/pendente/falhou), link para PDF
2. Se não há faturas (trial), mostra empty state: "Nenhuma fatura ainda. As faturas aparecerão aqui após o primeiro pagamento."
3. Método de pagamento atual visível: tipo (Visa/Mastercard/etc) + últimos 4 dígitos
4. Se não há método de pagamento (trial), mostra: "Nenhum método de pagamento cadastrado"
5. Banner verde "Plano atualizado com sucesso!" quando URL tem `?success=true`
6. Banner amarelo "Checkout cancelado. Seu plano não foi alterado." quando URL tem `?canceled=true`
7. Link "Gerenciar pagamento" abre Stripe Customer Portal
8. `pnpm typecheck`, `pnpm lint` e `pnpm test:run` passam sem erros novos

## Dev Notes

**Stripe Invoice object (campos relevantes):**
```typescript
{
  created: number;          // Unix timestamp
  amount_paid: number;      // Em centavos (BRL)
  status: 'paid' | 'open' | 'void' | 'uncollectible';
  invoice_pdf: string;      // URL do PDF
  hosted_invoice_url: string; // URL da fatura online
}
```

**Stripe Payment Method:**
```typescript
{
  card: {
    brand: 'visa' | 'mastercard' | 'amex' | 'elo';
    last4: string;
    exp_month: number;
    exp_year: number;
  }
}
```
