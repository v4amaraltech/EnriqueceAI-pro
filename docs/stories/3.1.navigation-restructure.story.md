## Complexity Estimate

**XL (8 points)** — Substituição completa do sistema de navegação (sidebar → top bar).

## Risks

- **R1:** Navegação top bar pode quebrar layout mobile existente
- **R2:** Rotas existentes dependem do sidebar layout — regressão potencial
- **R3:** Breadcrumbs contextuais exigem tracking de hierarquia de rotas

## Story

**As a** SDR ou manager,
**I want** uma navegação horizontal (top bar) com menus dropdown,
**so that** a experiência seja idêntica ao Meetime Flow e facilite o acesso rápido a todas as seções.

## Scope

**IN:**
- Top bar horizontal com dropdowns (Prospecção, Ligações, Estatísticas)
- Menu mobile (hamburger)
- Breadcrumbs contextuais
- Sub-bar com ações rápidas (Ligar, Usuário)
- Remoção da sidebar atual

**OUT:**
- Conteúdo das páginas de destino (outras stories)
- Módulo de Ligações (Wave 4)
- Módulo de Estatísticas (Wave 4)
- Animações/transições avançadas

## Acceptance Criteria

1. Top bar horizontal substitui completamente a sidebar
2. Dropdown "Prospecção" com submenus: Execução, Atividades, Cadências, Leads, Ajustes
3. Dropdown "Ligações" (placeholder para Wave 4)
4. Dropdown "Estatísticas" (placeholder para Wave 4)
5. Área direita: ícone notificações (bell), help (?), avatar do usuário com dropdown
6. Responsivo: hamburger menu em mobile com drawer
7. Breadcrumbs contextuais abaixo da top bar
8. Sub-bar com botão "Ligar" e link "Usuário"
9. Rota `/settings` acessível via Prospecção > Ajustes
10. Nenhuma rota existente quebrada (zero regressão)
